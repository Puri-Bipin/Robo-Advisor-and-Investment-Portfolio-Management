# Robo-Advisor-and-Investment-Portfolio-Management

Welcome to the Robo-Advisor-and-Investment-Portfolio-Management System! This system provides investment recommendations based on user inputs, risk tolerance, and investment criteria. It utilizes historical stock data and generates charts to visualize stock performance.

## Features

- Investment Recommendations: Based on user inputs such as risk tolerance, investment amount, and investment duration, the system generates investment recommendations for stocks.
- Historical Stock Data: Retrieves historical stock data using the Yahoo Finance API to calculate average return and volatility.
- Portfolio Performance: Calculates the performance of a user-defined portfolio based on the selected stocks.
- Interactive Charts: Generates bar and line charts to visualize stock performance for each recommendation.

## Technologies Used

- Python
- Flask: A web framework used for building the application.
- HTML: Markup language used for creating the web pages.
- CSS: Stylesheet language used for styling the web pages.
- JavaScript: Scripting language used for dynamic behavior on the web pages.
- yfinance: Python library for accessing and retrieving stock data from Yahoo Finance.
- matplotlib: Python library for creating visualizations, used for generating charts.

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/Puri-Bipin/investment-recommendation-system.git
   
2. Install required dependencies
    pip install -r requirements.txt
3. Run the Flask application:
    python app.py

Open a web browser and navigate to http://localhost:5000 to access the System.

Usage
Open the application in a web browser.
Enter your risk tolerance, investment amount, and investment duration.
Click on the "Get Recommendations" button.
View the investment recommendations along with average returns and charts.
Explore the portfolio performance section to see the performance of your selected stocks.
